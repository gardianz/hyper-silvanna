# proxy.txt — daftar HTTP proxy untuk bot (opsional)
# Satu URL per baris. Kosongkan / set proxy.enabled=false di config.json
# untuk paksa direct connection.
#
# Format yang didukung:
#   http://user:pass@host:port
#   http://host:port
#
# Bot pilih proxy deterministik per akun (1 akun = 1 proxy stabil).
# Mapping disimpan di proxy-assignments.json (auto-generate).
#
# Contoh:
# http://user1:pass1@104.207.59.11:3129
# http://user1:pass1@45.3.40.15:3129
